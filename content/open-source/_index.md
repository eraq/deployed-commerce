A collection of WordPress plugins and modifications I wrote between 2010 and 2012, primarily targeting WordPress MU and early WordPress 3 multi-site installs. Most were compatibility updates to existing plugins that had not kept pace with the WP3 multi-site changes, plus a couple of original tools built out of necessity for client work.

Source code for each project is archived on GitHub.
